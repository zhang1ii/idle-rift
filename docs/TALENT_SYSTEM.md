# 狂怒战士首版天赋树

## 设计目标

天赋用于改变技能组合和资源循环，不替玩家自动操作，也不让力量、精通、急速、暴击和全能失去原本意义。

- 绝大多数技能仍完整享受急速。
- “急速转能力强度”只允许出现在少数指定技能的特殊节点。
- 天赋只能在战前调整，首版免费洗点，鼓励针对普通层和 Boss 改 Build。
- 每个节点 1 点，共三条路线、四层、12 个节点。
- 可以跨路线混点，但一套配置只能选择一个第四层终极节点。

## 天赋点来源

天赋点只由新的守关 Boss 首杀提供：

```text
第5层Boss首杀：第1点
第10层Boss首杀：第2点
第15层Boss首杀：第3点
……
第35层Boss首杀：第7点
```

- 普通挂机、角色经验和装备提升不提供天赋点。
- 已击败守关 Boss 的重复击杀不再提供天赋点。
- 天赋点永久保留，免费洗点只退回已获得点数，不会增加总点数。
- 当前天赋树上限为 7 点；达到上限后继续击败守关 Boss 不再增加天赋点。
- 第5层 Boss 必须在没有天赋的状态下击败，其首杀奖励用于第6层之后的构筑。

这种来源让天赋成长与有限的关卡突破绑定，避免通过无限挂机获得无限能力。

## 解锁结构

| 层级 | 解锁要求 | 定位 |
|---:|---:|---|
| 1 | 无 | 建立路线基础属性 |
| 2 | 已投入2点，并点出本路线前置 | 改变核心技能 |
| 3 | 已投入4点，并点出本路线前置 | 建立循环联动 |
| 4 | 已投入6点，并点出本路线前置 | 终极节点，三选一 |

获得7点后，玩家可以点满一条四层路线，再投入另外三点形成副路线，但无法同时获得两个终极节点。

## 血痕路线

定位：流血伤害、持续恢复、对流血目标的终结爆发。

1. **刻骨裂痕**：流血伤害提高15%。
2. **血迹铭刻**：鲜血回响转化比例由75%提高到90%，治疗上限由35%提高到40%最大生命。
3. **饥渴伤口**：流血伤害的8%转化为治疗。
4. **血潮终结**：泄怒技能命中流血目标时，额外结算剩余流血伤害的40%，但不提前移除流血。

适合DOT、回复和单体Boss持久战，也能减少普通层药水消耗。

## 狂潮路线

定位：怒意生成、爆发窗口和泄怒伤害。

1. **沸腾血性**：撕裂打击额外获得5点基础怒意，之后仍受精通提高。
2. **连锁爆发**：狂怒爆发强化技能数由3提高到4。
3. **精准倾泻**：单体与范围泄怒技能伤害提高15%。
4. **无尽狂潮**：爆发强化期间释放泄怒技能，返还实际消耗怒意的20%。

适合追求击杀速度和硬狂暴前输出，但缺少额外生存能力。

## 铁壁路线

定位：生命、护盾稳定性，以及把正确承受的伤害转回怒意与反击输出。

1. **厚重筋骨**：最大生命提高8%。
2. **稳定怒意**：仅怒意壁垒失去急速冷却收益，每1%急速转化为1%壁垒护盾；其他技能仍正常享受急速。
3. **怒盾回流**：壁垒首次吸收伤害时返还本次消耗怒意的20%。
4. **不动如山**：记录壁垒吸收伤害的40%，使下一次泄怒技能附加等量伤害，附加值最多不超过100%攻击强度。

适合重击 Boss 和高战损挂机。它仍以部分直接伤害换取稳定性，但能通过怒意回流和承伤反击维持硬狂暴前所需的输出，而不是只延长失败时间。

怒意壁垒护盾总量上限为最大生命值，不能在长战斗中无限叠加。壁垒不会消耗“狂怒爆发”的强化次数，因为它不享受爆发的攒怒或泄怒强化。

## 配置示例

- 血痕主路线：血痕4点 + 狂潮前2层 + 厚重筋骨。
- 狂潮主路线：狂潮4点 + 血痕前2层 + 厚重筋骨。
- 铁壁主路线：铁壁4点 + 血痕前2层 + 沸腾血性。

## 明确取消的方向

不设计“角色所有急速全部转化为伤害、治疗和护盾”的通用天赋。急速必须继续作为影响出手速度与技能冷却的核心高级属性。只有类似稳定怒意这样的单技能特殊节点可以改变局部规则。

## 实现边界

当前阶段已实现天赋数据、守关 Boss 首杀点数、7点上限、层级、前置、终极节点互斥、战前锁定和公式查询。铁壁路线四个节点与狂潮前三层“沸腾血性、连锁爆发、精准倾泻”已经接入战斗原型并有自动测试；正式天赋树界面、配装预设，以及血痕4个节点和狂潮终极节点“无尽狂潮”的完整战斗接入仍待制作。
