# 第一裂隙 1～5 层玩法闭环

## 目标

第一裂隙用于验证完整核心循环：推进普通层、承受持续战损、战前消耗药水、获得装备、调整五技能顺序、被 Boss 卡住、farm 后完成首次突破并获得第1点天赋。

第5层 Boss 必须在没有天赋点的状态下击败。首杀后获得第1点天赋，从第6层开始参与构筑。

## 状态流程

```text
选择已解锁普通层
  → 战前配置五技能循环并检查生命
  → 必要时自动使用药水
  → 自动战斗（战斗中禁止药物与换技能）
  → 首次累计击杀20只
  → 永久通关并解锁下一层
  → 第4层通关后主动挑战第5层Boss
  → 胜利：解锁第6层并获得第1点天赋
  → 失败：返回普通层继续farm
```

普通层死亡是挂机终点：保留掉落、货币、材料、药水余量、通关进度和楼层解锁，但不会自动复活或继续模拟。

## 普通层规则

- 每个普通层分别保存首次通关击杀进度。
- 首次累计击杀达到20后通关；切换楼层不会清空进度。
- 通关后该层可以无限重复farm，不再重复触发解锁奖励。
- 普通敌人死亡后英雄不免费回满生命。
- 当前敌人死亡到下一敌人入场之间为战前窗口，可以自动使用药水。
- 当前敌人存活期间禁止使用任何药物。
- 普通层和离线挂机都不会提供天赋点。

## 五技能循环

- 进入战斗后技能选择与顺序锁定，只能退出到战前准备阶段修改。
- 每次出手机会从当前游标开始，按1→2→3→4→5循环扫描。
- 冷却未完成、资源不足或自身前置条件不满足的技能会被跳过。
- 释放首个可用技能后，下一次从它的下一个槽位继续扫描。
- 五个技能均不可用时，本次出手机会等待，不重置循环游标。
- 防御技能不读取Boss下一招，也不因敌方蓄力自动触发。

## 战前自动药水

```text
战斗中使用：禁止
普通战前目标生命：80%
每瓶恢复：40%最大生命
使用方式：战前连续使用，直到达到80%或药水耗尽
Boss入场：免费回满，不消耗药水
```

药水可以由怪物直接掉落、商店购买或材料制作。商店价格随当前裂隙阶段和累计购买数量逐渐提高。

## Boss入场、失败与奖励

Boss挑战由玩家主动发起。入场时免费回满生命、职业资源归零、技能冷却完成，并清除普通层临时增益、DOT和护盾；不消耗药水。

Boss战中禁止使用药物。失败不损失装备、材料、药水或通关进度。

第5层Boss首杀获得第1点天赋；重复击杀第5层Boss不会再次获得。之后第10、15、20、25、30、35层守关Boss首杀各提供1点，天赋点上限为7。

## 楼层教学

- 第1层裂隙野兽：基础攻击，教学五技能队列和怒意循环。
- 第2层腐血猎犬：最多3层流血，教学持续战损与恢复。
- 第3层灰甲守卫：护甲/减伤，教学易伤、DOT和高倍率泄怒。
- 第4层裂地蛮兽：蓄力重击，教学用纯技能顺序安排护盾和资源。
- 第5层裂隙守卫：无天赋综合检查迟缓、恫吓、重击、外骨骼和地板硬狂暴。

## 验收指标

- 战斗过程中无法使用药物或更换技能。
- 技能按五格循环执行，防御技能不读取敌方行动自动触发。
- 普通层死亡停止挂机并明确显示原因。
- 第5层Boss首次胜利只授予1点天赋，重复胜利为0。
- 普通挂机、离线收益和经验都不能产生天赋点。
