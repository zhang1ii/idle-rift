# 五技能回路系统

## 设计目标

Idle Rift 的自动战斗不只比较总战力，还让玩家编写一段会被执行、也可能被 Boss 破坏的“五步程序”。技能选择决定程序能做什么，顺序决定它是否稳定，急速、冷却、怒意、装备和敌方机制决定它如何变形。

- 六选五与排序必须是持续玩法，而不是一次配置。
- Boss 直接考验循环结构，而不是只增加生命和伤害。
- 传奇装备可以奖励运行结果，也可以改写执行规则。

## 1. 完整回路

只有依次成功释放第 1～5 格且没有任何跳过，才完成一次完整回路。

- 完成回路立即获得 15 怒意，然后重新记录。
- 冷却、资源、前置或裂化造成的跳过会断链。
- 实际释放顺序不是预期下一格会断链。
- 回路进行中五格全不可用也会断链。
- 普通层与 Boss 层使用同一规则。

## 2. Boss 技能格裂化

第 5 层 Boss 每次碎地板还会裂化一个技能格，使该格下一次被扫描时必定跳过一次。顺序固定为：2 → 4 → 1 → 5 → 3。

裂化只影响一次扫描，不永久删除技能。固定顺序允许玩家观察、重排和配装解决，不采用完全随机破坏。

## 3. 传奇的两种职责

传奇戒指或饰品保留两条普通词缀，并额外携带一个循环特效。同名特效不可重复生效。

### 3.1 结果型传奇

| 特效 | 部位 | 规则 |
|---|---|---|
| 裂隙节拍器 | 饰品 | 完整回路后，下一次攻击额外造成该次实际伤害 50% 的回响伤害 |
| 断链齿轮 | 戒指 | 断链积累 1 层，下一次攻击每层额外造成 20% 伤害，最多 3 层并在命中后清空 |
| 血色闭环 | 戒指 | 完整回路时额外结算剩余流血伤害的 30%，不移除流血 |

结果型传奇改变“完整或断裂以后得到什么”。

### 3.2 规则型传奇

| 特效 | 部位 | 规则 |
|---|---|---|
| 孤鸣核心 | 戒指 | 释放攻击时，其他 4 格每有 1 格当前不可释放，伤害提高 12%，最多 48%；冷却、资源、前置和裂化均计入 |
| 无源炉心 | 饰品 | 战前五格没有任何攒能技能时，耗能技能无视资源门槛并视为满额释放；其自身冷却仍然生效 |
| 裂隙熔接器 | 戒指 | 裂化格不再只被丢弃；其 70% 核心效果熔接到下一次实际释放技能，随后两格恢复正常 |
| 反震装甲 | 饰品 | 碎地板发生时若英雄仍有护盾，取消本次技能格裂化，并让下一次攻击额外造成 50% 伤害 |

规则型传奇改变“程序怎样执行”。裂隙熔接器当前以“后格完整效果 + 裂化格 70% 核心效果”实现：攻击附加伤害，攒能附加怒意和流血，爆发附加 70% 层数，治疗转化 70% DOT 记录，壁垒转化 70% 当前怒意。

## 4. 组合方向

- `无源炉心 + 孤鸣核心`：纯泄能与长冷却制造技能稀缺，用剩余技能打爆发。
- `裂隙熔接器 + 孤鸣核心`：裂化既提供融合技能，也提高当前不可释放数量。
- `反震装甲 + 裂隙节拍器`：按顺序提前开盾，保住完整回路并把防守转成反击。
- `裂隙熔接器 + 断链齿轮`：不追求绝对稳定，把裂化融合和断链蓄力都转为收益。

## 5. 失败诊断

- 地板全部破坏：报告输出不足、完整回路、断链、裂化打断和最常跳过技能。
- 生命归零：报告生存不足；若死于重击，补充壁垒时机与吸收量。
- 诊断说明事实，不自动替玩家重排技能或给唯一答案。

## 6. 原型边界

当前垂直切片的初始背包提供 7 件 T4 传奇测试物品，方便合作者立即验证组合。它们不是正式新存档奖励，也不代表最终掉率。接入存档与正式掉落表时必须移除展示赠送。

下一轮数值验证应覆盖：

- 无传奇、单传奇、两件组合和四件规则传奇的第 5 层击杀时间。
- 孤鸣核心是否鼓励合理空窗而不是纯粹停摆。
- 无源炉心是否让急速有价值但不会淘汰正常怒意流。
- 熔接不同技能类型时的 70% 换算是否公平。
- 反震装甲是否要求有意识地安排壁垒，而不是长期护盾常驻。
