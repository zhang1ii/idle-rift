# Idle Rift 职业战斗系统

## 已确认的设计规则

### 基础属性

- 力量
- 敏捷
- 智力
- 耐力

职业使用力量、敏捷或智力中的一种作为主属性。耐力提高最大生命。
力量系、敏捷系和智力系装备相互独立，不跨主属性混用。

### 高级属性

- 精通：效果由职业定义，强化该职业的核心机制。
- 急速：提高出手速度，并加快技能冷却恢复。
- 暴击：提高非固定暴击技能的暴击概率。
- 全能：提高伤害与治疗，同时降低受到的伤害。

装备在对应主属性与耐力之外，随机获得一至两种高级属性。

### 职业资源

每个职业拥有独特资源。五技能循环可以由以下职责组成：

- 资源生成技能
- 易伤或其他状态技能
- 自身增益技能
- 资源消耗技能
- 防御技能

合理的技能顺序应形成资源生成、状态铺垫、爆发消耗和防御承伤的循环。

### 技能调度

- 战斗中玩家不手动释放技能。
- 玩家在战前配置五个技能的顺序。
- 每次出手机会从当前队列位置开始向后扫描。
- 冷却未完成或资源不足的技能会被跳过。
- 找到第一个可用技能后释放，并从它的下一个位置继续下一轮扫描。
- 如果五个技能都不可用，本轮不释放技能。
- 普通攻击是否独立存在，后续确认。

攻击技能可以共享一致的基础冷却。防御技能根据强度设置不同冷却。急速同时缩短出手间隔和冷却恢复时间。

### 技能联动类型

- 易伤：使后续攻击获得增伤。
- 流血或其他 DOT：在一段时间内持续造成伤害。
- 固定暴击：特定技能必定暴击。
- 背击：满足背后攻击条件时获得收益，具体判定待确认。
- 自身减伤：在预期敌方爆发前进入防御状态。
- 资源联动：生成、强化和消耗职业资源。

## 当前力量系原型

当前实现仅用于验证系统，不代表最终职业名称与数值：

1. 战意斩：生成战意。
2. 破势打击：生成战意并施加易伤。
3. 裂伤斩：生成战意并施加流血 DOT。
4. 终结重击：消耗战意，固定暴击，并受到精通强化。
5. 铁壁架势：消耗战意，使下一次受到的伤害降低。

## 当前原型公式

- 出手间隔与技能冷却恢复速度使用 `1 + 急速百分比` 作为倍率。
- 暴击伤害暂定为普通伤害的两倍。
- 全能每 1% 提高 1% 伤害与治疗，并降低 0.5% 受到伤害。
- 精通暂时等比例提高力量系泄资源技能的伤害。

这些是便于验证循环的临时公式，正式数值和递减规则将在职业与装备系统稳定后确定。

## 待确认

- 首批职业的名称、主属性、资源和精通效果。
- 普通攻击是否独立于五技能队列持续发生。
- 背击在固定站位横版战斗中的触发条件。
- DOT 是否受急速影响跳速，以及是否采用属性快照。
- 高级属性是否需要等级换算、软上限或递减收益。
