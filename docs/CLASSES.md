# Idle Rift 职业设计

## 职业差异标准

每个职业必须拥有不同的战斗节奏、核心资源、装备词缀需求、套装机制和挂机画面表现。职业不是基础属性模板。

## 铁誓卫

- 核心资源：守势。
- 基础机制：受到攻击时有基础格挡概率；格挡减免伤害、积累守势并立即反击。
- 资源爆发：守势达到 100 后，下一次普通攻击自动变为裂盾猛击。
- 装备方向：护甲、格挡概率、反击伤害、低生命减伤。
- 定位：高层稳定推进，低层清图速度一般。

## 烬刃客

- 核心资源：杀意。
- 机制方向：高速连击、暴击、流血和击杀加速。
- 装备方向：攻击速度、暴击、流血层数、击杀触发。

## 裂隙咏者

- 核心资源：共鸣。
- 机制方向：元素技能循环，积累共鸣后自动释放范围法术。
- 装备方向：冷却、元素伤害、范围和技能等级。

## 灰烬牧灵

- 核心资源：残魂。
- 机制方向：敌人死亡产生残魂，自动召唤或强化随从。
- 装备方向：召唤数量、召唤伤害、持续时间和死亡触发。

首个完整制作职业为铁誓卫。其他职业先保留数据定义，等待战斗底层稳定后逐个实现。
